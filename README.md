# 🌦️ Weather Predictor

A responsive weather predictor website built using HTML, CSS, and JavaScript. This project allows users to search cities and view real-time weather conditions with a modern user-friendly interface.

---

## 📌 Problem Statement

People need quick access to weather updates for travel, planning, and daily activities. This project provides a simple web-based solution to check weather conditions instantly.

---

## 🎯 Objectives

* Search weather by city name
* Display temperature, humidity, and wind speed
* Show weather conditions with icons
* Create responsive UI for all devices
* Improve frontend development skills using APIs

---

## ✨ Features

✅ Search any city weather
✅ Real-time weather updates
✅ Temperature display
✅ Humidity and wind speed info
✅ Weather icons based on conditions
✅ Responsive modern design
✅ Easy-to-use interface
